## Installation
1. npm init
2. npm i express cors body-parser mongodb
3. npm i -D nodemon
4. change package.json as below:
5. "scripts": {
    "start": "node server/index.js",
    "dev": "nodemon server/index.js"
  },

6. mkdir server touch index.js
7. npm i -g @vue/cli
8. vue create client
9. cd client
10. npm i axios
